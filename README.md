# 🌽 AI Early Warning Mobile Application

An AI-powered mobile application developed to support maize farmers in Tanzania through real-time weather monitoring, climate risk assessment, pest prediction, and yield forecasting.

## Features

- 🌤 Real-time weather monitoring
- 🌍 English and Kiswahili language support
- 🌧 Rainfall forecasting
- 💧 Vapour Pressure Deficit (VPD) assessment
- 🐛 Fall Armyworm (FAW) risk prediction
- 🌱 AI crop health assessment
- 📅 Seasonal crop monitoring
- 🌽 Machine learning-based maize yield prediction
- 🤖 Smart farming recommendations
- 💾 Automatic weather history storage

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- XGBoost
- SQLite
- OpenWeather API
- Pandas
- NumPy
